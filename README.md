# 🏀 NBA Analytics

Portfolio project focused on sports analytics and data visualization using Power BI.

## Overview
This repository contains a collection of data analysis projects focused on the NBA.

The goal is to explore how basketball has evolved over time using data — from changes in playstyle and scoring efficiency to player and team-level trends.

I work with a large historical dataset (CSV) covering NBA data from 1947 to the present day.

As a basketball fan, combining the sport with data analysis makes working on this project especially engaging.

---

## Projects

👉 [3PT Evolution](https://github.com/karolwalaszczyk1989/nba_analytics/tree/main/3pt-evolution)

### 1. 3PT Evolution
Analysis of how the importance of the 3-point shot has changed across decades.

Key findings:
- Over 10x increase in 3-pointers made per game since the 1980s  
- Shift from defense-oriented basketball (90s / early 2000s) to modern offensive play  
- Significant growth in 3-point shot share
